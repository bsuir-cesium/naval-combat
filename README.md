# naval-combat

## Тест-кейсы

### Проверка смерти корабля

Одинарный, полностью/частично раненый:
- У центра поля;
- В 4-ёх углах;
- У границ поля.

Неодинарный, полностью/частично раненый, вертикаль/горизонт:
- У центра поля;
- В 4-ёх углах;
- У границ поля.

### Проверка победы

Поле:
- Победа одного из игроков;
- Отсутствие победы
